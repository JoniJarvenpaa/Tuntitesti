1. git pull  Vetää remotesta mahdolliset muutokset omalle koneelle
2. tehdään tarvittavat muutokset/päivitykset, AKA. töitä
3. git add <muutokset, tai * jos kaikki>  --> Lisää tiedostot seuraavaan versioon
4. git commit -m "Kerrotaan mitä muutoksia on tullut tehtyä"
5. git push       --> Vie pätsin remoteen, eli githubiin.

when in doubt, git status
